TO'G'RILASH BO'YICHA QISQA YO'RIQNOMA

1) client papkasida eski .next va node_modules papkalarini o'chiring.
2) Terminal ochib client papkaga kiring.
3) npm install yozing.
4) npm run dev yozing.
5) Brauzerda http://localhost:3000 ni oching.
